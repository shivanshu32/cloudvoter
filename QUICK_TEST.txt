TROUBLESHOOTING:

Port 5000 in use?
  → netstat -ano | findstr :5000
  → taskkill /PID <PID> /F

Module not found?
  → venv\Scripts\activate
  → pip install -r requirements.txt

Frontend won't start?
  → npm install
  → npm start
